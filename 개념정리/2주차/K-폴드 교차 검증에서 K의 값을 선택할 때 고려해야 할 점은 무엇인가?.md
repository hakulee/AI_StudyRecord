# K-Fold 교차검증
- 데이터를 K개로 나누고 K번 훈련 & 검증 반복하는 방법

데이터를 K개로 나누는 겻은 작업자가 설정

K를 설정하는 대략적인 기준:
- 데이터 크기에 따라 나눔
  => 데이터 양이 적을수록 K는 많을수록 좋음
데이터가 적으니 폴드를 많이 늘려 많이 학습할 수 있도록 함

- 부하가 많이 걸려 데이터가 많을스록 K를 작게 설정하는 것이 좋음
> K를 5~10정도로 우선 돌려보고 돌아가는 상황 보고 추가 조정

방법
데이터를 K개로 나눔
그 중 하나는 테스트, 나머지는 훈련에 사용
이걸 K번 반복해 결과를 내 평균으로 모델 성능 평가
최적의 파라미터 찾음
